Configuration
=============

Create tax branch each operating unit,

* Go to *Settings > Users & Companies > Tax Branch Operating Units*
* Create new tax branch code for master data
* Go to *Settings > Users & Companies > Operating Units*
* Select Tax Branch in Operating Unit

Usage
=====

Tax Invoice
===========

Add a Branch to be included in the VAT recording (including undue) and allow users to edit it

Withholding Tax
===============

The `WHT Certificates` menu has been updated to include Branch selection, and users are required to select the Branch themselves.
