Usage
=====

To use this module, you need to:

- Go to Invoicing -> Reporting -> Withholding Tax Reports.
    - Select income tax form(pnd3, pnd53).
    - Create "Date Range".

* View - Report type html.
  Export PDF - Report type pdf.
  Export XLSX - Report type xlsx.
